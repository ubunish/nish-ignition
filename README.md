# nish-ignition

Flag-driven robotics workstation installer for macOS and Ubuntu. Installs apps and packages, then clones and wires Nish's own repos.

The name nods to First Motive — first motion, ignition.

```bash
git clone https://github.com/ubunish/nish-ignition.git ~/nish-ignition
cd ~/nish-ignition
./setup.sh
```

Full usage, flag reference, and uninstall land in a later step. See `docs/` for the original hand-written macOS and Ubuntu guides.
